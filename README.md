# 🎮 Fun Games for Kids

A modern collection of interactive browser-based games designed specifically for young children (ages 5+). Built with vanilla HTML, CSS, and JavaScript featuring a beautiful, accessible design system.

## ✨ Features

- 🎨 **Modern Design System** - Clean, professional UI with consistent styling
- 📱 **Fully Responsive** - Works perfectly on desktop, tablet, and mobile
- ♿ **Accessible** - Designed with accessibility best practices
- 🚀 **Fast & Lightweight** - No external dependencies, loads instantly
- 🏠 **Easy Navigation** - Home button in every game for seamless experience

## 🌐 Live Demo

**Play now:** [https://y-maeda1116.github.io/games/](https://y-maeda1116.github.io/games/)

## 🎯 Games Collection

### 🧩 **Puzzle & Memory Games**
- **🧩 Matching Game** - Find pairs of matching cards to test your memory
- **🧩 Jigsaw Puzzle** - Drag and drop pieces to complete beautiful pictures
- **🔍 Spot the Difference** - Find all differences between two similar images

### 🎨 **Creative Games**
- **🎨 Drawing Game** - Digital canvas with brushes, colors, and fun stamps
- **👗 Dress-Up Game** - Style characters with different clothes and accessories

### 🎵 **Action & Rhythm Games**
- **🎵 Rhythm Game** - Press keys to the beat and create music
- **🔨 Whack-a-Mole** - Classic arcade game with quick reflexes
- **📚 Learning Quiz** - Educational questions to learn while playing

### 🎲 **Strategy Games**
- **⚫ Othello** - Classic board game with strategic thinking
- **♟️ Shogi (将棋)** - Japanese chess for advanced players

## 🚀 Quick Start

### For Players
1. Visit the [live demo](https://y-maeda1116.github.io/games/)
2. Click on any game card to start playing
3. Use the 🏠 Home button to return to the main menu

### For Developers
```bash
# Clone the repository
git clone https://github.com/y-maeda1116/games.git

# Open in browser
cd games
open index.html
```

## 🛠️ Technology Stack

- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Styling**: CSS Custom Properties, Flexbox, Grid
- **Fonts**: Inter (Google Fonts)
- **Icons**: Unicode Emojis
- **Hosting**: GitHub Pages

## 🎨 Design System

- **Colors**: Modern palette with CSS custom properties
- **Typography**: Inter font family for readability
- **Spacing**: Consistent spacing scale (0.25rem - 3rem)
- **Components**: Reusable button, card, and input styles
- **Responsive**: Mobile-first approach with breakpoints
